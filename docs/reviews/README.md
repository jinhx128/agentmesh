# Review Archive

This directory stores committed review records that are useful outside the local
`.agentmesh/` run packet history.

- `studio/`: Studio, desktop shell, Tauri/Electron, DMG, and frontend stack
  architecture reviews.
- `workflow/`: workflow/runtime implementation phase reviews and release-gate
  records.

Local workflow-backed reviews still originate in `.agentmesh/runs/<run-id>/`.
Only copy a review here when it should become durable project documentation.
