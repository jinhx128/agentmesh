# AgentMesh v0.1.11

## Highlights

- Studio/Desktop 使用新的银灰浅色界面、青蓝主操作与统一状态反馈，并补齐目标窗口尺寸下的响应式和可访问性细节。
- Desktop 使用单一 SVG 母版生成的新 Converge 应用图标，并保持既有 Tauri 资产路径不变。
- Desktop 可在 Settings / About 检查、下载、验证并安装后续签名更新。
- Desktop 可检测、安装和更新 PATH 可见的 npm CLI，无需用户输入安装路径。
- Agent Integrations 保留 Codex、Claude Code、Cursor、Antigravity 和 OpenCode，已移除 Copilot 集成。

## Upgrade notes

- 从 v0.1.10 升级必须手动用本版本 DMG 替换一次 `AgentMesh.app`；安装 v0.1.11 后，后续版本可使用应用内更新。
- macOS DMG 仍为未签名、未 notarize 的 Apple Silicon 版本，首次打开可能需要右键 Open，或在 System Settings / Privacy & Security 中允许。
- CLI 是独立渠道，请使用 npm 或同版本 tarball 安装 `@jinhx128/agentmesh@0.1.11`；更新 CLI 不会替换 Desktop App。
