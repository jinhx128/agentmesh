# AgentMesh v0.1.12

## Highlights

- 修复 macOS 从 Finder、Dock 或 `open` 无参数启动 Desktop 时可能长期停留在 “Starting AgentMesh” 的问题。
- Desktop 不再递归同步扫描 Home 目录来猜测工作区；现在直接读取 AgentMesh workspace registry，选择最近启用且仍存在的工作区，并在没有记录时安全回退到 Home。
- `run`、`flow run` 和被记录的 `call` 新增可选 `--title`；主控 Agent 可生成 4–24 字中文标题，Runtime 在缺省时使用确定性的 `工作区名-摘要` 或 `工作区名-HH:mm:ss`，技术 ID 保持不变。
- 保留 v0.1.11 的银灰 Studio UI、Converge 图标、应用内 updater、CLI 安装/更新与 Copilot 移除等完整能力。

## Upgrade notes

- 从 v0.1.10 升级或 v0.1.11 停在启动页时，请用本版本 DMG 直接替换 `AgentMesh.app`；若 v0.1.11 可以打开，也可从 Settings / About 使用应用内更新。v0.1.12 安装后，后续签名版本可继续使用应用内更新。
- 无参数启动只选择 workspace registry 中的最近有效项目；尚未注册的项目可先运行 `agentmesh workspaces add /path/to/project`，或使用 `AGENTMESH_STUDIO_WORKSPACE=/path/to/project` 固定工作区。
- macOS DMG 仍为未签名、未 notarize 的 Apple Silicon 版本，首次打开可能需要右键 Open，或在 System Settings / Privacy & Security 中允许。
- CLI 是独立渠道，请使用 npm 或同版本 tarball 安装 `@jinhx128/agentmesh@0.1.12`。
