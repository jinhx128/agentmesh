# AgentMesh v0.1.13

## 修复

- 修复 Desktop 主 WebView 导航到随机端口 loopback App Server 后，Tauri capability 仍只允许 bundled local context，导致 Settings / About 的原生“检查应用更新”在网络请求前被 ACL 拒绝的问题。
- main capability 仅允许 `http://127.0.0.1:*` 使用既有 updater/restart 权限；不开放 localhost、其他 IP、HTTPS 或公共 remote origin，也不增加 filesystem/shell 权限。
- 原生 updater 检查、下载或重启失败时保留经过脱敏和限长的 Tauri 错误，不再把所有非 HTTP 错误统一显示为 `Network request failed`。

## 兼容与升级

- packet schema、call schema、CLI 参数、SDK/API 和 workspace 数据均不变；这是 `0.1.12` 的兼容补丁版本。
- 已安装 `0.1.12` 的用户需要用 `0.1.13` DMG 手动替换一次 `AgentMesh.app`，因为 `0.1.12` 的原生 updater capability 会在发起更新请求前拒绝调用。安装 `0.1.13` 后，后续签名版本可继续从 Settings / About 更新。
- CLI 与 Desktop 是独立渠道；CLI 可从 npm 或同版本 GitHub tarball 安装。
