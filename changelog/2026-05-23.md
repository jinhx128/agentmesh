# 2026-05-23

## 16:18 - 发布 0.1.1 并修复 Studio 安装态工作区读取

- 版本统一提升到 `0.1.1`：同步根包、workspace 包、Tauri/Cargo 版本、package lock、运行时兼容版本、MCP client version、Desktop updater 示例元数据、README 和首页发布资产说明。
- Desktop Studio 在 Finder / LaunchServices 启动且 cwd 为 `/` 时，不再把根目录或空的 `$HOME/.agentmesh` 当作 workspace；支持 `AGENTMESH_STUDIO_WORKSPACE`，并会从常见开发目录发现最近的 AgentMesh workspace。
- Tauri sidecar 启动参数透传 `--workspace` / `--workspace=...`，并把 macOS `Info.plist` 权限说明纳入发行包校验，避免安装态访问用户目录时缺少 usage description。
- Studio 运行详情修复 workflow flow 尺寸和方向箭头，统一 workflow 与产物流节点尺寸；节点和详情展示实际 Agent，退出码缺字段时显示“未记录”，current-only 阶段显示“无外部进程”。
- 审查发布 raw review tab 优先展示 Agent 名称，设置页兼容性元数据缺失时展示可诊断的“尚未生成 / 未记录”状态。
