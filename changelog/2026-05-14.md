## 14:53 - L5.7 Run Single-Writer Lock

- 新增 `packages/runtime/src/packet/lock.ts`，用 run 目录内
  `.agentmesh.lock/lease.json` 实现 single-writer mutation lock。
- `flow attach`、`flow dispatch`、`flow retry`、`flow resume`、
  release-check prompt refresh 和 `release-check summary --write` 现在会先获取
  run mutation lease，保护 `status.json`、`events.jsonl`、`artifacts.toml` 和
  stage output writes。
- active lease 会给出 `run is locked` 诊断，包含 operation、pid、expires_at 和
  lock path；expired lease 会被回收后继续执行。
- expired lease 回收会先创建 reclaim marker 并二次确认 lease 仍过期，避免并发回收时
  误删新 lease。
- 新增 `docs/contracts/run-lock.md`，明确 `.agentmesh.lock/lease.json` 不是
  packet artifact，只是本地互斥机制。
