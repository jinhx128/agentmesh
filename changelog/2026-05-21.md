# 2026-05-21

## 23:08 - 替换 Antigravity CLI 并收紧模型语义

- Agent adapter 矩阵移除 Gemini CLI，新增 `antigravity-cli` / `antigravity`，命令默认使用 `agy`；Skill target、Desktop 集成、Studio 资源分组、SDK 元数据和相关文档同步为 Antigravity。
- Antigravity 注册不再假装支持 per-Agent 模型参数；`agy` 当前 print 模式没有模型 flag，Agent 配置统一写入 `model = "current"`，表示使用 Antigravity CLI 当前选中的模型。若 CLI 传入具体模型，会提示 warning 并存储为 `current`。
- Antigravity 模型探测改为从 `agy` 可执行文件和本机用户数据里发现实际模型标签，只用于判断工具是否可用；Studio 创建 / 编辑 Agent 时对 Antigravity 仅展示 `current`。
- Studio Agent 创建 / 编辑弹窗预加载所有支持工具的模型；工具下拉改为全量工具可见，只有对应工具模型未就绪时禁用单项，避免当前工具未就绪时无法切换到其它可用工具。
- 高级设置里的“允许自动分发”和“需要用户确认”补充问号说明，说明改为悬停展示；开关只响应开关本体点击，不再整行切换。
