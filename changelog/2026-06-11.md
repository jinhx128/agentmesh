## 10:35 - Studio 全局 workspace 历史可见性

- 新增用户级 workspace registry，使用 `~/.config/agentmesh/workspaces.json` 记录已登记 workspace，并用 realpath 后的路径生成稳定 `workspace_id`。
- 新增 `agentmesh workspaces add/list/enable/disable`，支持导入旧 workspace；`agentmesh call` 产生记录时会 best-effort 登记当前 workspace。
- Studio Runs 与 Calls 默认聚合当前 workspace 和 registry 中 enabled workspace 的历史记录，并在列表、详情、artifact preview、call adoption 链路中贯穿 `workspace_id`。
- Studio 前端列表展示 workspace label，并用 `workspace.id + run_id/call_id` 做选中 key，避免跨 workspace 同名记录串联。
- 详情、artifact preview 和 adoption API 只允许通过当前 workspace 或 registry 中 enabled workspace 解析记录，拒绝直接按任意 path 读取本机文件。
